# 파이썬 기초 정리

## 변수초기화 방법

변수(Variable)

- 데이터를 저장할 수 있는 메모리 공간의 이름
- 쉽게 데이터에 액세스할 수 있도록 특정메모리에 이름을 붙인 것
- 지속적으로 내용을 다른 값으로 갱신할 수 있다.
  <br />

| 용어     | 정의                                            |
| -------- | ----------------------------------------------- |
| 변수     | 데이터를 저장할 수 있는 메모리 공간의 이름      |
| 데이터형 | 컴퓨터에서 다루어지는 다양한 데이터 값들의 유형 |

<br />

```python
number = 123  # 변수 생성 및 사용 문장
number = number + 123  # 변수 사용 문장
number = None  # none
pi = 3.14  # float 변수
comp = 1 + 2j  # complex 변수
isValid = True  # bool 변수
msg = 'massage'  # str 변수
arr = [0, 1, 2, 3, 4, 5]  # list 변수
```

<br />

> - 파이썬에서는 모든 자료가 클래스로부터 만들어진 객체
> - 변수는 다른 객체를 참조하는 참조자 또는 포인터의 역할을 한다.

<br />

## 연산자

### 산술 연산자

| 연산자 | 의미                   | 사용 예      | 결과 값 |
| ------ | ---------------------- | ------------ | ------- |
| \+     | 덧셈                   | a = 5 + 8    | a = 13  |
| \-     | 뺄셈                   | a = 5 – 3    | a = 2   |
| \*     | 곱셈                   | a = 5 \* 7   | a = 35  |
| /      | 나눗셈                 | a = 3 / 5    | a = 0.6 |
| //     | 몫만 구하는 나눗셈     | a = 8 // 5   | a = 1   |
| %      | 나머지만 구하는 나눗셈 | a = 8 % 5    | a = 3   |
| \*\*   | 제곱                   | a = 3 \*\* 2 | a = 9   |

<br />

### 대입연산자

| 연산자 | 의미                  | 사용 예   | 결과 값                       |
| ------ | --------------------- | --------- | ----------------------------- |
| =      | 대입                  | a = 5     | 변수 a에 정수 5의 값을 대입   |
| +=     | 덧셈 후 대입          | a += 5    | a = a + 5 와 동일한 연산식    |
| -=     | 뺄셈 후 대입          | a -= 5    | a = a - 5 와 동일한 연산식    |
| \*=    | 곱셈 후 대입          | a \*= 5   | a = a \* 5 와 동일한 연산식   |
| /=     | 나눗셈 후 대입        | a /= 5    | a = a / 5 와 동일한 연산식    |
| //=    | 몫을 구한 후 대입     | a //= 5   | a = a // 5 와 동일한 연산식   |
| %=     | 나머지를 구한 후 대입 | a %= 5    | a = a % 5 와 동일한 연산식    |
| \*\*=  | 제곱을 구한 후 대입   | a \*\*= 5 | a = a \*\* 5 와 동일한 연산식 |

<br />

### 관계연산자

- 2개의 값이나 변수 간의 관계를 연산
- 결과는 True 또는 False

| 연산자 | 의미        |
| ------ | ----------- |
| \>     | 크다        |
| <      | 작다        |
| \>=    | 크거나 같다 |
| <=     | 작거나 같다 |
| ==     | 같다        |
| !=     | 같지 않다   |

<br />

### 논리연산자

- 2개의 값이나 변수 간에 논리적인 연산을 수행

| 연산자 | 의미                                                      |
| ------ | --------------------------------------------------------- |
| and    | 두 개의 피연산자 중에 하나라도 `False`이면 결과도 `False` |
| or     | 두 개의 피연산자 중에 하나라도 `True` 이면 결과도 `True`  |
| not    | 피연산자 값의 반대값으로 전환                             |

<br />

### 연산자 정리

- 나눗셈 연산자 변경
  - 연산자 / : 실수의 나눗셈 (결과가 실수)
  - 연산자 //: 정수 연산(floor division)
- 이항 연산자 \*\* 추가
- 단항 연산자 ++, -- 제공 xx
  - `x++이 아니라 x += 1`
- 관계 연산자 >, <, >=, <=, ==, !=
- 논리 연산자: `||, &&, !` -> `or, and, not`
- in과 not in 연산자

```python
'a' in 'banana'  # True
'seed' in 'banana'  # False
```

## 조건문

조건문 예시

```python
if 조건식:
  코드 1
else:
  코드 2
```

<br />

### if~elif~else문

```python
age = int(input('나이 입력: '))
price = 20000

if 1 <= age < 6 :
  print('입장료 무료 입니다!')
elif 6 <= age < 60 :
  print(f'입장료는{price}원 입니다!')
elif age >= 60 :
  print(f'입장료는{price*0.5}원 입니다!')
else :
  print('1살 이상의 나이를 입력해 주세요!')
```

<br />

### if~in~else문

- **in** 명령어는 **단어가 문장에** 들어있는 경우 `True`, 아니면 `False` 반환

```python
pocket = ['스마트폰', '신용카드', '현금']
if '현금' in pocket:
  print('택시를 타세요')
else:
  print('걸어가세요')
```

<br />

### 중첩 조건문 예제

- if 문 안에 if문을 중첩하여 표한한 중첩 조건문

```python
id = input('아이디 입력: ')
if id == 'python':
  pwd = input('비밀번호 입력: ')
if pwd == 'a1234':
  print('로그인 성공')
else:
  print('비밀번호 오류')
else:
  print('아이디 오류')
```

## 반복문

- 반복문의 조건식 끝에는 항상 `:` 이 있어야된다.
- if문과 마찬가지로 공백으로 종속 코드들을 포함한다.

1. while
   - 조건식의 결과가 True 또는 False 인지에 따라 코드를 반복 실행하는 구조
   - while 문의 조건식들은 논리 연산이 가능한 문장 즉, True, False 판별이 가능해야 된다.
2. for
   - for 문에 `range()` 함수를 이용하여 시작값부터 종료값 까지 증가값 만큼씩 증가하여 <br />
     리스트의 원소를 하나씩 꺼내 변수에 저장한 뒤, for문에 종속된 코드들을 실행
   - range(0, 10, 1)과 range(0, 10)과 range(10) 은 모두 0부터 9까지 1씩증가로 원소를 하나씩 꺼내 반환하는 for문

> range(start = 0, end, step = 1) <br />
> state: 시작값, end: 종료 값, end 값은 미포함, step: 증가되는 값, 기본값은 1

<br />

### while

```python
# 0을 입력하면 종료
num = None
while num != '0'
  print('while문으로 반복중...')
  num = input('번호를 입력하세요(종료 0): ')
print('while문 종료')
```

<br />

### for

```python
for i in range(1, 6):
  print('*' * i)

# 또는

for i in range(5):
  print('*' * (i + 1) )
```

실행결과

```
*
**
***
****
*****
```

### 기타제어문

- break <br />
  반복문 안에서 사용하면 반복문을 `강제 종료`
- continue <br />
  반복문 안에서 사용하면 **남은 부분 건너 띄고** 반복문 처음으로 돌아감
- pass
  - 아무것도 하지 않는 명령어
  - `무한루프` 또는 `임시함수`를 만들 때 당장 아무 것도 작성하지 <br />
    않으면 에러가 나므로, `임시용`으로 사용

## 문제로 총정리

q: 1부터 n까지 정수의 합을 구하는 프로그램

```python
print('1부터 n까지 정수의 합 출력')
n=int(input('n: '))
total = 0
# 2번째 인자에 +1 해준다
for index in range(1, n+1):
  total += index
print(f'1부터 {n}까지 정수의 합은 {total}입니다.')
```

실행결과

```
1부터 n까지 정수의 합 출력
n: 10
1부터 10까지 정수의 합은 55입니다.
```

<br /><br />

q: 연속하는 정수의 합을 구하는 프로그램

```python
print('a부터 b까지 정수의 합 출력')
a=int(input('a: '))
b=int(input('b: '))
total = 0
# 두 정수를 비교해서 분기처리
if (a < b):
  for index in range(a, b + 1):
    total += index
elif (b < a):
  for index in range(b, a + 1):
    total += index
print(f'{a}부터 {b}까지 정수의 합은 {total}입니다.')
```

실행결과

```
a부터 b까지의 정수의 합 출력
a: 1
b: 10
1부터 10까지 정수의 합은 55입니다.
```
